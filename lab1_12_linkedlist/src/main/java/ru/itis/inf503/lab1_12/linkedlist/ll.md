1. Создать класс LinkedList503, хранящий элементы в виде связного списка
2. Реализовать внутренний класс Node, описывающий узел
3. add(int value) - добавление нового элемента со значением value
в конец списка
4. add(int value, int n) - добавление нового элемента со значением value
   в позицию n списка
5. int pop() - удалить последний элемент и вернуть его значение
6. int remove(int n) - удалить элемент в позиции n и вернуть его значение
7. int size() - вернуть количество элементов
8. Протестировать в отдельном классе
