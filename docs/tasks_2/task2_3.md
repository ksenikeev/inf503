# Map (карта, отображение, "dict")

1. Набор пар ключ:значение (K,V)
2. ключи - уникальны
3. если добавляем новую пару с уже существующим ключем - значение заменяе

Map503
1) put(K,V)
2) get(K) -> V or null
3) size()
4) remove(K) -> V or null
5) clear()
6) interface Entry (K,V)
7) entrySet() -> set of Enrty
8) keySet() -> set of Keys
9) values() -> list of values
