# Обработка исключений

1. try catch finally
2. Исключение - класс, имеющий родителем Exception
3. throws - индикатор в методе о необходимости обработки перечисленных исключений

```java
try {
    
    // блок кода, содержащий вызовы которые могут привести к выбросу исключения
    
} catch (Exception1 e) {
    // обрабатываем 1-е исключение
} catch (Exception2 e) {
    // обрабатываем 2-е исключение
    // Тип 2-го исключения не должен наследоваться от типа 1-го исключения (такие исключения покроет обработка 1-го исключения)
} finally {
    // код который будет выполнен в любом случае (произойдет или нет выброс исключения)
}
```
