# Методы (функции) в языке Java

```java
public class Test { // 1
    public static void main(String args[]) { // 2
        // ... наш код
    }
    
}
```

## Определение
```java
    [Модификатор_доступа] [static] тип_возвращаемого значения названиеМетода([аргументы]) {
        // операторы метода
        return возвращаемое_значение;
    }

```
```java
    public static int summ(int a, int b) {
        return a + b;
    }   
```
