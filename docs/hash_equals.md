# Контракт между equals и hashCode


1. Контракт между hashCode() и equals():

 - Если два объекта равны согласно методу equals(), то их хеш-коды должны быть одинаковыми.

 - Если два объекта имеют одинаковый хеш-код, они не обязательно должны быть равными по equals().

2. Cвойства хеш-функции:
 - Детерминированность: один и тот же объект должен возвращать один и тот же хеш-код.

3. Типовые реализации:
```java
    hash = Objects.hash(field1, field2, field3);

    hash = 31 * field1.hashCode() + field2.hashCode();
    hash = 31 * hash + field3.hashCode();
    
    hash = 17*field1.hashCode()
```
