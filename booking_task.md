# JSON

описывает пары "ключ":значение, перечисляемые через запятую

json объект заключается в скобки { ... }

внутри описывается набор пар "ключ": значение

Ключ - строка-идентификатор

значение:
- json объект
- строка, дата (в кавычках)
- число (без кавычек)
- массив ([ ... ])

Пример:
```json
{
  "id": 19, 
  "name": "student",
  "group": {
    "number": "11-503",
    "institute": "ИТИС"
  },
  "bith_date": "2007-11-01",
  "lang": ["русский","татарский","английский"]
}
```
